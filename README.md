# E-Commerce Data Cleaning Project

## About

This is my first data cleaning project, completed as part of my DecodeLabs Data Analyst Virtual Internship.

In this project, I cleaned and explored an e-commerce dataset using Python and Pandas.

## What I Did

- Loaded the dataset
- Checked the dataset information
- Checked for missing values
- Checked for duplicate records
- Examined unique values in categorical columns
- Reviewed numerical data using summary statistics
- Replaced missing values in the `CouponCode` column
- Removed extra spaces from text columns
- Saved the cleaned dataset

## Tools Used

- Python
- Pandas
- VS Code
